Frequently Asked Questions

1. What is the diffrence between var, let and const?
var is function-scoped, let and const are block-scoped, with const being immutable.
2. What is the diffrence between map(), forEach() and filter()?
map() returns a new array, forEach() doesn't return anything, and filter() creates a new array with filtered items.
3. What are arrow functions and how are diffrent from regular functions?
Arrow functions have a concise syntax and lexically bind this, unlike regular functions.
4. How do JAvascript Promises work?
JavaScript Promises represent an eventual value, either resolved or rejected, and handle asynchronous operations.
5. How does closure work in javascript?
Closure in JavaScript allows inner functions to access variables from their outer functions, even after the outer function has executed.
